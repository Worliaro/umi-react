### 基于umi搭建React快速开发框架 01

### 介绍
基于umi搭建一个快速开发框架，react 应用框架。umi 以路由为基础的，支持类 next.js 的约定式路由，以及各种进阶的路由功能，并以此进行功能扩展，比如支持路由级的按需加载。
我们会在基于umi的基础上，开发出一个框架通用功能和业务功能
